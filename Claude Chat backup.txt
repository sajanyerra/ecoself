### PROJECT OVERVIEW
A React web app for personalised sustainability recommendations
tailored to life in Ireland. Portfolio project.
Minimalist, fun, nature-themed design.
Privacy-first — all data stored in localStorage only.
No backend, no accounts, no data collection.

### TECH STACK
- React + Vite
- Tailwind CSS v3
- Framer Motion (animations)
- Recharts (installed, not used yet)
- React Router DOM (routing)
- Groq API — llama-3.3-70b-versatile (AI recommendations)
- html2canvas (score card download)
- localStorage (all user data)
